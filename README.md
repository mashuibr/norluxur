# NORLUXUR - Luxury Fashion E-Commerce

A sophisticated React + Vite website for a luxury clothing brand featuring exclusive pre-orders, live auctions, and a museum-like shopping experience.

## üé® Features

### Core Functionality
- **Pre-order System**: Timed releases with countdown timers and availability tracking
- **Live Auctions**: Real-time bidding system for one-of-a-kind pieces
- **Archive Collection**: Curated vintage pieces available by appointment
- **Private Viewing**: Booking system for personalized shopping experiences
- **Membership Tiers**: Silver, Gold, and Platinum membership programs

### Design Elements
- **Museum-like Aesthetics**: Elegant, minimalist design with luxury typography
- **Custom Cursor**: Interactive cursor that enhances the premium feel
- **Smooth Animations**: Fade-ins, slide-ups, and parallax effects
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile
- **Preloader Animation**: Elegant entry experience

### Technical Features
- **React 18**: Modern component-based architecture
- **Vite**: Lightning-fast build tool and dev server
- **CSS Modules**: Component-scoped styling
- **State Management**: React hooks for interactive features
- **Countdown Timers**: Real-time countdown for releases and auctions
- **Modal System**: Product details and bidding modals
- **Toast Notifications**: User feedback system

## üöÄ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and visit: `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## üé® Design Philosophy

The website embodies luxury through:
- **Typography**: Cormorant Garamond for headings (elegance), Montserrat for body (readability)
- **Color Palette**: Deep blacks (#0a0a0a, #1a1a1a) with gold accents (#d4af37)
- **Spacing**: Generous white space and breathing room
- **Animations**: Subtle, sophisticated transitions
- **No Discounts**: Maintains brand exclusivity by avoiding any promotional pricing
